# CloudLux Website

Официален статичен сайт за CloudLux – луксозни облачни тавани.

## Live URL

https://sviievmla.github.io/CouldCelling/

## Основни файлове

- [index.html](index.html) – основна страница за деплой
- [CloudCelling.html](CloudCelling.html) – работно копие на страницата
- [robots.txt](robots.txt) – инструкции за crawler-и
- [sitemap.xml](sitemap.xml) – карта на сайта
- [site.webmanifest](site.webmanifest) – PWA/brand метаданни
- [assets/images/gallery/cloud-ceiling-kit.png](assets/images/gallery/cloud-ceiling-kit.png) – hero/OG изображение

## SEO и техническа настройка

- Meta description, keywords, canonical, hreflang
- Open Graph + Twitter Card мета тагове
- Schema.org (`LocalBusiness`, `WebSite`, `FAQPage`)
- `robots.txt` + `sitemap.xml`
- Lazy loading на изображенията в галерията
- Mobile-first оптимизации и достъпност (`skip link`, semantic landmarks)
